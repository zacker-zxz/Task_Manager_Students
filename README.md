# NexusTask — Student Task Manager

NexusTask is a clean, minimal, and fully-featured student task management application designed to help you organize assignments, exams, and personal goals.

## Features
- **Task Tracking**: Easily create and categorize tasks (Assignments, Exams, Personal).
- **Persistent Storage**: Node.js and Express backend with a lightweight JSON file database ensures your tasks are saved.
- **User Authentication**: Secure signup and login system utilizing encrypted passwords and session tokens.
- **Responsive UI**: A modern, dark-themed interface built using HTML, CSS (with glassmorphism and animations), and Vanilla JavaScript. 

## How to Run Locally

1. **Install Dependencies**
   Make sure you have Node.js installed, then run:
   ```bash
   npm install
   ```

2. **Start the Server**
   ```bash
   npm start
   ```

3. **Open the App**
   Navigate to [http://localhost:3000](http://localhost:3000) in your web browser.
